Docker Teste

Basta utilizar o seguinte comando :

docker container run -p 8080:80 -v $(pwd)/html:/usr/share/nginx/html nginx

Depois disso só será preciso fazer o uso do servidor local e poderá visualizar esta página em html.

Com o comando abaixo o acesso é feito de forma imediata e utilizando o container:

docker container run -d --name ex-daemon-basic -p 8080:80 -v $(pwd)/html:/usr/share/nginx/html nginx

Para verificar quais containers estão rodando basta rodar o seguinte comando:

docker container ps

Para parar o container que está rodando acima basta executar o seguinte código:

docker container stop ex-daemon-basic

3 comandos interessantes:

docker container start ex-daemon-basic

docker container restart ex-daemon-basic

docker container stop ex-daemon-basic

Comandos para avaliar algumas características do seu container

docker container logs ex-daemon-basic

docker container inspect ex-daemon-basic

Comando para verificar que tipo de sistema operacional está sendo executado dentro do seu container:

docker container exec ex-daemon-basic uname -or
